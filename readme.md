# 🕴️ Barney Bot – Suit Up Your Chat! 😎🍸


## Hey you! Yes, YOU. Welcome to the ultimate chat experience.

Barney Bot isn’t your average boring chatbot.  
It’s **Barney Stinson from ‘How I Met Your Mother’**, teaching you how to **suit up**, **be awesome**, and throw in some legendary confidence in your life.  

---

## 🏆 Features – Because mediocrity is not allowed

- 💬 **Chat with Barney** – short, punchy responses with arrogance and style.  
- 📚 **Important Words System** – save your phrases and let Barney sprinkle them in conversation.  
- 🎯 **Learning Chat** – reinforce your vocabulary while getting life lessons (Barney style).  
- 🔥 **Commands and Buttons** – `/AddWords`, `/phrases`, `/exam`, `/LearningOn`, `/LearningOff`  
- 💾 **Persistent Memory** – conversations and important words are saved in `history.json`.  

---

## ⚡ How to Run – Make it legen… wait for it… dary

1. Clone the repo:

```bash
git clone https://github.com/yourusername/barney-bot.git
cd barney-bot
````

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Create a `.env` file with your **Telegram Token** and **HuggingFace API Key**:

```env
TOKEN=your_telegram_token_here
API_KEY=your_huggingface_key_here
```

4. Start the bot:

```bash
python app.py
```

5. Open Telegram, find your bot, and type `/start` –
   **suit up**, baby! Buttons and commands will guide you through the legendary experience.

---

## 🎩 Commands You’ll Use

| Command        | What it does                                  |
| -------------- | --------------------------------------------- |
| `/AddWords`    | Add important words/phrases to your list      |
| `/phrases`     | Show all your saved words                     |
| `/exam`        | Test yourself in a legendary vocabulary exam  |
| `/LearningOn`  | Turn learning mode on (Barney-style learning) |
| `/LearningOff` | Turn learning mode off                        |
